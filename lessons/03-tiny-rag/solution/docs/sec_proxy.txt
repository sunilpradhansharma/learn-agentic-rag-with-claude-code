A DEF 14A, commonly called a proxy statement, is a document that public companies must file with the SEC before a shareholder meeting. It discloses information that allows shareholders to make informed voting decisions, including proposals to be voted on, executive compensation details, director biographies, and the identities of major shareholders. The "DEF" stands for "definitive," meaning the final version approved for distribution to shareholders. Proxy statements are a primary source of data for governance researchers and institutional investors evaluating board composition and pay practices.
